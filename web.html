BaraMoto Página Web

Tecnologías :


Página web informativa y administrativa, creada para la empresa de venta de refacciones “BaraMoto”, diseñada para informar eventos, consejos y capacitaciones a todos los colaboradores de la empresa.


La página cuenta con enlace a base de datos del programa de escritorio, para facilitar el registro de nuevos usuarios, así como piezas nuevas, con posibilidad de generar reportes de ventas, o consultas de existencias.


Estructura en HTML5 con diseño de elementos en CSS3, los elementos de advertencias, así como la validación de formularios de datos, soportados por el lenguaje de programación JavaScript.


Características

  • Inicio de sesión

  • Ingreso de piezas de inventario

  • Reporte de ventas

  • Consulta de existencias

  • Registro de nuevos productos

  • Registro de nuevos usuarios

  • Información relevante
